# Environment variables the Guidewire suite needs at build/run time.
# This file lists NAMES ONLY — never values (some are secrets).
#
# Tags:
#   [set]       - non-sensitive; the skill can help set these (known/derivable values)
#   [user-show] - user-set, NOT secret; checker shows its value so you can verify/change it
#   [user]      - SECRET or account-specific; YOU must obtain & export. Value never shown.
#
# DEPLOYMENT_ID is the one devs change routinely: swap the environment segment (e.g. the
# "qa3" in ...:dev:qa3:/deployment/... ) to whatever env you want to mirror (dev2, etc.).
#
# (Separately, run the server in the local environment via -Denv=local — that is a JVM
#  property set in the gwb command / Studio run config, not an exported env var.)
